Phenotype network

In phenotype network, each node represents a phenotype. Phenotype network is an aggregation network from individual dataset networks, where two nodes are connected by an edge whenever a dataset of phenotype A has been found similar to a dataset of phenotype B in any dataset network of the same species. Edge weight represents the frequency of the similarity (A,B).

Dataset network

In dataset networks, each node represents a dataset. Edges represent similarities between datasets, as distances between two empirical, multivariable, statistical distributions, intuitively implying that molecular quantities measured in the two datasets are inter-correlated in a similar way, i.e., the covariance matrices are similar. Computation in these networks is unsupervised and depends only on the omics data distributions, ignoring any annotation or labeling of each dataset.
